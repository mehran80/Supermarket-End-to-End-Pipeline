# Supermarket End-to-End Data Pipeline

## 🚀 Project Overview
This project is a complete Data Engineering pipeline that cleans raw supermarket data, loads it into a PostgreSQL database, and visualizes key business insights in Power BI.

## 🛠️ Tech Stack
- **Database:** PostgreSQL (Storage)
- **Language:** Python (ETL Process - Pandas, SQLAlchemy)
- **BI Tool:** Power BI (Visualization)

## 📊 The Pipeline
1. **Extraction:** Loads raw data from a 10,000-row CSV.
2. **Transformation (Silver Layer):** Cleans dates, handles missing values, and removes duplicates using Python.
3. **Analytics (Gold Layer):** Created 5 specific reports (Monthly Trends, Category Performance, Shipping Efficiency, etc.).
4. **Visualization:** Connected Power BI to the Gold tables to build an interactive dashboard.

## 📈 Key Insights
- Corporate clients have a higher average spend than individual consumers.
